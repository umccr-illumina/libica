    libica.openapi.v2.BundleDataApi¶

    All URIs are relative to /ica/rest

    Method HTTP request Description
    get_bundle_data GET /api/bundles/{bundleId}/data Retrieve the list of bundle data.
    link_data_to_bundle POST /api/bundles/{bundleId}/data/{dataId} Link data to this bundle.
